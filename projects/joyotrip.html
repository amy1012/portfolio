01 OVERVIEW

Solving "Decision Paralysis" in Travel Planning.

JoYoTrip is a hyper-personalized recommendation system that combines Large Language Models (LLMs) with real-time data. Unlike traditional apps that offer static suggestions, JoYoTrip dynamically adapts itineraries based on weather, user interactions, and real-world constraints.

05 SYSTEM ARCHITECTURE

RAG-Based Recommendation Engine

The core of JoYoTrip is a Retrieval-Augmented Generation (RAG) framework powered by LangGraph. It ensures that the LLM's creativity is grounded in real-world data.

  • Frontend: Native Android (Kotlin + Jetpack Compose).
  • Backend: Python FastAPI for high-concurrency handling.
  • Logic: LangGraph nodes for breaking down planning steps (Preference -> Validation -> Optimization).

System Flow: Integration of LLM, Google Maps API, and Weather Data.

06 SYSTEM FEATURES

The JoYoTrip Experience

1. Hyper-Personalized Itinerary

Leveraging RAG technology, the system analyzes user preferences and real-time data to construct tailored travel plans. It goes beyond generic suggestions, ensuring every spot aligns with the user's unique intent and constraints.

2. Intelligent Chat Assistant

Acts as a knowledgeable travel companion. It maintains conversational context memory, understanding complex requests and refining itinerary details through natural dialogue interaction.

3. Dynamic Weather Adaptation

Integrated with real-time weather APIs. If rain or adverse weather is forecast, the system proactively suggests indoor alternatives and instantly reroutes the itinerary to ensure a smooth experience.

07 SYSTEM ADVANTAGES

Key Technical Strengths

Accurate Information (RAG + API)

Knowledge sources are reliable and traceable. By integrating Google Maps & OpenWeather APIs with RAG, we eliminate LLM hallucinations.

Contextual Memory Focus

Unlike generic LLMs that lose context, our system maintains a continuous memory of the trip theme, ensuring seamless adjustments.

One-Stop Planning Interface

A unified interface designed specifically for travel. It allows for efficient planning and instant response to changes, reducing decision fatigue.

09 TAKEAWAYS

Project Reflections

Challenges Overcome

Balancing the creative freedom of LLMs with the need for factual accuracy was a major hurdle. Implementing the LangGraph-based RAG structure was critical to grounding the AI's outputs in reality.

Impact & Future Scope

This project demonstrated the potential of hyper-personalized AI agents in reducing decision fatigue. Future iterations could focus on integrating multi-modal inputs (e.g., image-based preferences) and expanding real-time data sources like live traffic.
