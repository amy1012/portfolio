01 OVERVIEW

Sharing nature, connecting travelers.

NSBlog is a dynamic web platform designed for travelers who want to share their experiences in nature. The goal was to build a fully functional Content Management System (CMS) that allows users to easily publish stories, while providing readers with a clean, immersive reading environment.

The project emphasizes scannability and ease of use, ensuring that the beauty of the photos and stories takes center stage.

MY CONTRIBUTION

  • UI/UX Design: Designed the overall layout and visual style using CSS and Bootstrap, focusing on readability and contrast.
  • Frontend Development: Implemented the responsive grid system and integrated CKEditor for rich text editing.
  • System Integration: Collaborated with backend developers to connect the frontend forms with PHP/MySQL utilizing AJAX for smooth data handling.
02 DESIGN CONSIDERATIONS

Prioritizing Readability & Navigation

Intuitive Navigation

Adopted a simple, flat hierarchy. The navigation bar is persistent, allowing users to switch between "Home", "All Posts", and "Backend" seamlessly without getting lost.

Scannable Layout

Used a card-based layout for the blog feed. Large thumbnails and concise titles allow users to quickly scan multiple stories and find what interests them.

Clear Contrast

Selected a color palette inspired by nature (Earth Green & Cream White). High contrast between text and background ensures accessibility and readability for long-form content.

03 KEY FEATURES

Dynamic Content Management

Beyond the frontend, we built a robust backend system to handle data persistence and user management.

CRUD Operations

Create, Read, Update, Delete. Authorized users can manage their posts through a dedicated dashboard. We implemented image uploading logic that stores file paths in the MySQL database while saving the actual assets on the server.

Account Management

Administrators have access to an account management panel to oversee user activity. The system includes secure login/logout functionality using PHP sessions to protect backend routes.

Smart Search

Implemented a keyword-based search function that queries the database for matching titles or content, helping users quickly find specific travel destinations.
